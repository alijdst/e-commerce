# e-commerce
Sample shop for providing and selling products to customers with details.
The shop is connected to Iranian banks gateway and give two result to user, a successful buy and a unsuccessful buy. Also admin can check the buy process and tracking code with additional model(Payment) and default Iranian bank gateway model.
With custom user model (Account), users can create, sign-in, sign-out their account with custom fields. Can change password and if the user forgot his/her password, can revceive the paswword reset email.
The Cart model handle products that user selected to buy.
